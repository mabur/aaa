Detailed Description

This module defines the functions:

  • add
  • subtract
  • multiply
  • divide

They do elementwise arithmetic operations on containers. This module supports expressions on the general form binary_operation(input_left, input_right, output) where:

  • The binary_operation is one of add, subtract, multiply, divide.
  • The output is a container.
  • At least one of input_left and input_right is a container. The other could be a scalar.

We call these vector space operations since a mathematical vector space has the following elementwise operations defined:

  • addition of two vectors.
  • subtraction of two vectors.
  • multiplication of a vector and a scalar.
  • division of a vector and a scalar.

We represent mathematical vectors as containers.
